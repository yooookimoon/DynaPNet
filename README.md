# DynaPNet

Environment
The proposed model was developed and evaluated under the following environment:
Operating System: Ubuntu Linux
Python: 3.10.8
CUDA: 11.8
PyTorch: 2.1.2
TorchVision: 0.16.2
PyTorch Geometric: 2.7.0
NumPy: 1.26.4
SciPy: 1.15.3
Pandas: 2.3.3
Scikit-learn: 1.7.2
Matplotlib: 3.9.0
TensorBoard: 2.17.0

Installation Requirements
Install dependencies via:
pip install torch==2.1.2 torchvision==0.16.2
pip install torch-geometric==2.7.0
pip install numpy==1.26.4 scipy==1.15.3 pandas==2.3.3
pip install scikit-learn==1.7.2 matplotlib==3.9.0 tensorboard==2.17.0

Hardware
GPU: NVIDIA vGPU (48 GB memory)
CUDA: 11.8

Evaluation Metrics
The following metrics are used to evaluate model performance:
Accuracy (ACC)
Area Under the ROC Curve (AUC)
Sensitivity (SEN)
Specificity (SPE)
